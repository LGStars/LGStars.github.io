1 一维数组的定义与引用 - 考核概率80%

真考连接

其中一维数组的引用是考查的重点。

在操作题中，常在修改题中考察一维数组的定义

1.1 一维数组定义

1.1.1 一维数组是指数组中的每个元素只带有一个下标的数组。

1.1.2 执行完 long array[10]，语句之后，C编程将会在内存中连续开辟10个存储单元，每个存储单元的名称就是每个数组元素的名称，如array[0]...，可以直接用名字来引用各个存储单元。

1.2 一维数组的引用

1.2.1 引用形式：数组名[下标表达式]。

1.2.2 注意：

    （1）一个数组元素实质上是一个变量名，代表内存中的一个存储单元，一个数组占据的是一连串连续的存储单元。

    （2）数组下标可以是整型常量，也可以是整型表达式。

    （3）只能逐个引用数组元素而不能一次引用整个数组。

    （4）在C语言运行过程中，系统并不自动检查数组元素的下标是否越界，所以数组两端都可能因为越界而破坏其他存储单元中的数据，甚至破坏程序代码，因此，要保证数组下标不能越界。

2 二维数组的定义与引用 - 考核概率90%

2.1 二维数组的定义

2.1.1 在C语言中，二维数组元素排列的顺序是：按行存放，即在内存中先按顺序存放第一行的元素，再存放第二行的元素。也占据一块连续的内存单元。

    注意：在C语言中明确规定：在定义二维数组时，后一个下标值不能省略，否则无法判断某一行元素的具体个数。

2.2.2 二维数组的一般形式：

    类型说明符 数组名[常量表达式][常量表达式]

    例如： int c[3][4] (三行四列)

2.2 其元素的引用

2.2.1 二维数组的表达形式：

    数组名[下标表达式1][下标表达式2]

2.2.2 注意：

    （1）：数组的下标可以是整型表达式，如c[3-1][3*2-2]。

    （2）：数组的元素可以出现在表达式中，也可以被赋值，如：a[1][3]=b[3][2]/2。

2.3 二维数组的初始化

2.3.1 在定义二维数组的同时可以给二维数组的各个元素赋初值。

    例如：float m[2][2] = {{1.5,3.2},{0.8}};

    全部的初值放在一对花括号中，每一行的初值又分别放在一对花括号中，之间用逗号隔开。当某行一对花括号内的初值个数少于该行的元素的个数时，系统会自动的给后面元素赋初值0.同样不能跳过前面的元素给后面的元素赋值。

3 字符数组 - 考核概率20%

真考连接

选择题中主要以读程序题要求考生写出程序运行结果的形式出现，是必考内容.

在操作题中，使用数组处理字符是常考内容。

3.1 字符数组的定义

3.1.1 字符数组就是数组中的每个元素都是字符，定义方法是逐个对数组的元素进行赋值。

3.1.2 如果花括号中提供的初值个数大于数组长度，则编译时会按语法错误处理。如果小于数组长度，则将剩下的元素自动定为空字符（'\0'）。

3.2 字符串和字符串结束的标志

3.2.1 C语言中，为了测定字符串的实际长度，C语言规定了一个字符串结束标识，以字符'\0'代表。

3.2.2 系统对字符串常量也自动加一个'\0'作为结束标志。例如：

    char c[] = "c program";

    该字符中共有9个字符，但在内存中占用10个字符，最后一个字节是'\0'是由系统自动加上的。

3.2.3 注意：'\0'代表ASCII码为0的字符，是一个“空操作符”，它什么也不干。在输出时也不输出'\0'，它只是一个结束的标识。

3.3 字符数组的输入/输出

3.3.1 方法：

    （1）用“%c”格式符将字符逐个输入或者输出。

    （2）用“%s”格式符将整个字符串一次输入或者输出。

3.3.2 注意：

    （1）用格式符“%s”在输出字符串时，printf（）函数输出项应该是数组名，而不是数组中的某个元素，如 “printf（“%s”，c[0]）;”是不对的。

    （2）在使用scanf（）函数时，输入项是字符数组名，不用再加取地址符“&”。

3.4 字符串处理函数

在使用这些函数时，必须在前面的命令行包括标准头文件“string.h”。

3.4.1 函数名称 - 调用形式 - 作用 - 说明

    （1）puts() - puts(字符数组) - 以'\0'结束 - 输出的字符串中可以包含转义字符

    （2）gets() - gets(字符数组) - - 只能一次输入一个字符串

    （3）strcpy() - strcpy(字符数组1，字符数组2) - 把字符数组2中的内容复制到字符数组1所指的存储空间中。函数返回数组1的值。 - 字符数组1空间 > 字符数组2的空间

    （4）strcat() - strcat(字符数组1，字符数组2) - 该函数将字符数组2所指的字符串的内容连接到字符数组1所指的字符串后面，并自动覆盖字符数组1的末尾'\0'。该函数返回字符数组1的地址值。 - 字符数组1空间应该足够大。

    （5）strcmp() - strcmp(字符数组1，字符数组2) - 该函数用来比较字符数组1和字符数组2所指的字符串的大小。若字符数组1 > 字符数组2，函数值 > 0（正数）；若字符数组1 = 字符数组2，函数值= 0；若字符数组1 < 字符数组2，函数值 < 0（负数） - 根据字符的ASCII依次对字符数组1和字符数组2所指的字符串进行比较，当出现第一对不同的字符时，既由这两个字符决定所在串的大小

    （6）strlen() - strlen(字符数组) - - 这一长度不包括串尾的结束标识'\0'。
